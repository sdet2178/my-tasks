One-time Google setup

This app saves your tasks to a private file in your Google Drive. Complete the steps below once, then paste your Client ID to get started. You can collapse this panel afterwards.

1
Create a Google Cloud project

Go to console.cloud.google.com → click Select a project → New Project. Name it anything (e.g. "My Tasks App") and click Create.

2
Enable the Google Drive API

In your project, go to APIs & Services → Library, search for Google Drive API, and click Enable.

3
Configure the OAuth consent screen

Go to APIs & Services → OAuth consent screen. Choose External, fill in an app name (anything), your email, and click Save and Continue through all steps. Under Test users, add your own Gmail address.

4
Create an OAuth Client ID

Go to APIs & Services → Credentials → Create Credentials → OAuth client ID. Choose Web application. Under Authorised JavaScript origins, add the exact URL where you open this file (e.g. https://your-domain.com or http://localhost for local use). Copy the Client ID.

5
Paste your Client ID below

Your tasks will be saved to a private tasks.json file in your Drive's app data folder — invisible to other apps and not browseable in regular Drive.
