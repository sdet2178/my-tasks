One-time Google setup

This app saves your tasks to a private file in your Google Drive. Complete the steps below once, then paste your Client ID to get started.

1
Create a Google Cloud project

Go to console.cloud.google.com → click Select a project → New Project. Name it anything and click Create.

2
Enable the Google Drive API

Go to APIs & Services → Library, search for Google Drive API, and click Enable.

3
Configure the OAuth consent screen

Go to APIs & Services → OAuth consent screen. Choose Internal or External, fill in your app name and email. Under Test users, add your Gmail address.

4
Create an OAuth Client ID

Go to Credentials → Create Credentials → OAuth client ID. Choose Web application. Under Authorised JavaScript origins, add your GitHub Pages URL (e.g. https://your-username.github.io). Copy the Client ID.

5
Paste your Client ID below

Tasks will be saved to a private tasks.json in your Drive's app data folder.
